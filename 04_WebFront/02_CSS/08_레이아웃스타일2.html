요소의 여백(영역) 관련 속성

        HTML 요소는 총 4가지의 영역으로 구성 되어있다.

        1. content 영역
            - 요소의 내용이 작성되는 영역
            (시작태그 ~ 종료태그 사이에 작성되는 내용)

        2. padding 영역
            - content 영역과 border 영역 사이 

        3. border 영역
            - 요소의 테두리가 지정되는 영역
            - content보다 바깥쪽에서 content 영역을 감쌈

        4. margin 영역
            - 다른 요소와의 간격

padding, border, margin의 방향성

        padding, border, margin은 방향에 따라 크기를 별도 지정 가능

        방향은(상하좌우)
        
        - 영역 이름 : 값1;
            -> 상, 하, 좌, 우 모두 동일 크기 지정

        - 영역 이름 : 값1 값2;
            -> 값1 : top bottom
            -> 값2 : right left

        - 영역 이름 : 값1 값2 값3;
            -> 값1 : top
            -> 값2 : right left
            -> 값3 : bottom

        - 영역 이름 : 값1 값2 값3 값4;
            -> 값1 : top
            -> 값2 : right
            -> 값3 : bottom
            -> 값5 : left
            (시계방향 top, right, bottom, left)


        - top, bottom, left, right 이용

        영역이름-top    : 값;
        영역이름-bottom : 값;
        영역이름-left   : 값;
        영역이름-right  : 값;
